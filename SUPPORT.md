# Support

Fast path for users
- If something is broken or confusing, open an issue:
  https://github.com/laolaoshiren/hermes-agent-webui/issues/new/choose

What to include
- what you tried to do
- what you expected
- what happened instead
- screenshots if the problem is visual
- browser / OS / deployment details if relevant

Project response style
- ship fast
- get real users touching the product early
- fix visible problems quickly
- keep the public repo active and responsive
